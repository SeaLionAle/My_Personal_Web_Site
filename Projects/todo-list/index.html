JavaScript To-Do App

This is a functional task management application built entirely with vanilla JavaScript. It demonstrates DOM manipulation, event handling, and data persistence.

Tech Stack:

HTML5 CSS3 Vanilla JS LocalStorage

Features:

  • Add, check, and delete tasks dynamically.
  • Data is saved in your browser's LocalStorage.
  • Fully responsive UI with CSS variables.

Tips:

  • Type "Hello World!" in the input box.
  • Click the "Add" button to save it.
