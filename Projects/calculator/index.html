JavaScript Grid Calculator

A powerful, dual-mode calculator built with Vanilla JavaScript and modern CSS Grid. It features a dynamic cursor for editing expressions, ghost parentheses, and seamlessly switches between Standard and Scientific modes.

Tech Stack:

HTML5 CSS3 Vanilla JS CSS Grid

Features:

  • Standard & Scientific modes powered by dynamic CSS Grid.
  • Interactive cursor to edit expressions midway.
  • Full keyboard accessibility (Numpad, Enter, Arrows).
  • Smart auto-completion for missing parentheses.
